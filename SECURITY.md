# Security Policy

## Supported Versions

Currently only the latest version recives security fixes and only for the latest version security fixes are acceptet. 

## Reporting a Vulnerability

To report a security issue send an email to security@secureelections.software. 
We try to respond as fast as possible to your report and resolving the issue in a quick and uncomplicatet manner. 
A first response to your report can be expectet within 48h after you send the email.

Please alwayse make a Responsible Disclousure by sending a small prove of concept or some instructions on how to reproduce the issue to security@secureelections.software.
